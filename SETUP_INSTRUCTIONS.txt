METHOD 2: USING A LOCAL WEB SERVER (Recommended for development)
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

Option A - Python (if you have Python installed):
   1. Open terminal/command prompt
   2. Navigate to the unifind-project folder:
      cd path/to/unifind-project
   3. Run: python -m http.server 8000
   4. Open browser and go to: http://localhost:8000

Option B - Node.js (if you have Node.js installed):
   1. Install http-server globally: npm install -g http-server
   2. Navigate to the unifind-project folder
   3. Run: http-server -p 8000
   4. Open browser and go to: http://localhost:8000

Option C - VS Code (if you use VS Code):
   1. Install "Live Server" extension
   2. Right-click on index.html
   3. Select "Open with Live Server"
